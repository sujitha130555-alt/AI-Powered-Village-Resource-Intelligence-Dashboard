# AI-Powered Village Resource Intelligence Dashboard

## Overview

This project analyzes village resources and identifies villages requiring immediate attention using Artificial Intelligence and Data Analytics.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Power BI
* Scikit-Learn
* Julius AI
* Excel

## Features

* Resource Analysis
* Water Scarcity Detection
* Road Service Analysis
* Waste Management Analysis
* Internet Connectivity Analysis
* Priority Score Calculation
* AI-Based Village Classification
* Recommendation System
* Interactive Power BI Dashboard
